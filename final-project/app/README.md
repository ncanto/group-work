# Instructions

- Download the app folder and save it to your local machine.

- Open a terminal and navigate to the directory where the app folder is located.

- Type "python app.py" in the terminal and press Enter.

- Wait for the message "Running on http://127.0.0.1:5000" to appear in the terminal.

- Open a web browser and enter the URL http://127.0.0.1:5000 in the address bar. This will open the website in the browser.

- Alternatively, if the first method does not work, open the "Digital Tools for Finance's FINAL-PROJECT.html" file in a web browser to access the website.

- Explore the website to learn more about the project and its features.

- If you encounter any issues or have any questions, feel free to contact the project team for assistance.
